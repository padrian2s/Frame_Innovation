The Logic of Design: Creating What Does Not Yet Exist

Commentary

Summary

The design formula: WHAT + HOW → VALUE. In conventional problem-solving, WHAT and HOW are known; the question is about VALUE. In design, VALUE is known and WHAT and HOW must be discovered. This reversal is what enables genuine innovation.

Design Formula: WHAT (designed thing) + HOW (working principle / frame) → VALUE (desired outcome). In design, VALUE is known; WHAT and HOW must be discovered.

Key Patterns

Design reasoning is abductive and generative. The five practices (coevolution, problem situations, frames, themes, discourse) work as an integrated system. Expertise means comfort with complexity and ability to hold multiple frames.

Practical Applications

Practice abductive reasoning: work backward from desired outcomes. Instead of 'what can we do with what we have?' ask 'what would it look like if this were truly solved?' This reversal opens new possibilities.
