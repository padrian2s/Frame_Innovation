# Frame Innovation

Interactive study guide for **Frame Innovation** with simulators.

## Overview

A web-based interactive platform for exploring the concepts in the Frame Innovation book. Includes HTML reader pages, JavaScript-driven simulators, and a structured chapter-by-chapter layout with CSS styling.

## Contents

- `index.html` -- Main entry point
- `reader.html` -- Book reader interface
- `pages/` -- Individual chapter pages
- `simulators/` -- Interactive simulators for key concepts
- `css/` / `js/` -- Styling and scripts

## Getting Started

```bash
bash run.sh
```

Open `index.html` in a browser to begin exploring.
